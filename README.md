# TF Keyboard Calibration

Move /tf frames around using your keyboard!

## Usage:

    Manual alignment of camera to world CS:
    =======================================
    MOVE: X  Y  Z  R  P  YAW 
    ------------------------
    up    q  w  e  r  t  y 
    down  a  s  d  f  g  h 
    Fast: u 
    Med:  i 
    Slow: o 
    Save: p
